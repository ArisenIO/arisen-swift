ArisenRpcProvider

public class ArisenRpcProvider

Default RPC Provider implementation. Conforms to ArisenRpcProviderProtocol. RPC Reference: https://developers.rix.io/arisen-aos/reference

    The blockchain ID that all RPC calls for an active instance of ArisenRpcProvider should be interacting with.

    A node URL.

    Number of seconds to wait between each retry (default is 1 second).
